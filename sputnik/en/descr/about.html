About project

Sputnik project is a template project for developing real satellite or other data processing system based on alt_processing framework.

Project is based on my personal experience of creating real satellite data processing systems.

This code was rewritten specially for github and not used in real projects yet.

I would be glad any feedback, contribution and proposal to apply alt_processing in the real project.
